## Task 1 

You have a list of n items. 
 
e.g = [1 , 1, 4, 4, 6, 32, 32, 32, 5, 522, 0, .....] , [24 , 3, 5, 5, 6, 11, 11, 32, , 5 , 5, 52, 32, 0, -4, .....]
 
You need to find out the occurrence of each value from this list.
